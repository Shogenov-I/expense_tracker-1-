# Expense Tracker

**Автор:** [Шогенов Идар]

## Описание программы

Графическое приложение для учёта личных финансов. Позволяет добавлять расходы с указанием суммы и категории (еда, транспорт и т.д.), фильтровать их по категориям и датам (с помощью календаря), а также подсчитывать общие затраты за любой период. Все данные надёжно сохраняются в файл `expenses.json`.

## Как использовать

### Установка зависимостей

Для работы программы требуется библиотека `tkcalendar` для виджета календаря. Установите её командой в терминале:
`pip install tkcalendar`

### Запуск приложения

1. Убедитесь, что Python установлен.
2. Запустите файл из папки проекта командой в терминале:
   `python expense_tracker.py`
3. Интерфейс программы откроется автоматически.

### Основные функции

1.  **Добавление расхода:**
    *   Введите сумму в поле «Сумма».
    *   Выберите категорию из выпадающего списка.
    *   Выберите дату с помощью виджета календаря.
    *   Нажмите кнопку **«Добавить расход»**.
2.  **Фильтрация:**
    *   Выберите категорию в списке «Фильтр по категории».
    *   Нажмите кнопку **«Фильтровать»**.
3.  **Подсчёт суммы:**
    *   Выберите даты начала и конца периода.
    *   Нажмите кнопку **«Сумма за период»**.
4.  **Сохранение данных:**
    *   Нажмите **«Сохранить данные»**. Данные сохраняются в файл `expenses.json`.
    *   При следующем запуске программа загрузит данные автоматически.
    *   Для принудительной перезагрузки данных из файла используйте кнопку **«Загрузить данные»**.
